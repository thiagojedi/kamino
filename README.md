# Kamino

This is the Kamino, a Universal Blue based image for my personal use on a Inspiron 15 5584 from 2019.

The objective is to make it basically a reconstruction of Aurora, but with legacy Nvidia drivers. I'll be adding other dependencies as needed.

## Why don't I just use Aurora??

Aurora does not have legacy Nvidia support and recommends users towards Bazzite.

## Then why not Bazzite?

To me, it seemed easier to create a new thing than "clean up" a complex image like Bazzite. 

# How to use this image?

I'll not create an ISO, since this is for my personal usage. Sorry.

If you still want to test it, just install any Fedora Atomic image with KDE (e.g. Aurora or Fedora Kinoite), then switch to this image as so:

```
sudo bootc switch ghcr.io/thiagojedi/kamino
```
